Workshop 6: Generalized linear models

Chapter 2 Preparing for the workshop

To prepare for this workshop, you must do the following steps:

Download the R script and data required for this workshop:

  • R Script
  • Mites
  • Faramea

Make sure to load the following packages (see how in the R script):

  • ggplot2
  • lme4
  • MASS
  • vcdExtra
  • bbmle
  • DescTools
install.packages("ggplot2")
install.packages("lme4")
install.packages("MASS")
install.packages("vcdExtra")
install.packages("bbmle")
install.packages("DescTools")

library(ggplot2)
library(lme4)
library(MASS)
library(vcdExtra)
library(bbmle)
library(DescTools)
